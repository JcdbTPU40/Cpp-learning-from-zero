【第一章のまとめ】

■ 1. C++プログラムの基本構造
- ヘッダのインクルード：#include <iostream>, #include <string>, #include <ctime>, #include <cstdlib> など
- 名前空間：using namespace std;
- エントリポイント：int main() { ... }
- return 0; で正常終了（書かれていない場合もあるが意味は同じ）

■ 2. 画面への出力と文字列リテラル
- cout と挿入子 << を使って文字列や値を出力する。
  例: cout << "こんにちは\n";
- 文字列リテラル内で \n を使うと改行される。
- 複数の文字列リテラルは空白や改行をはさんで並べても連結される。
  例:
    "ABC"
    "DEF"
  は "ABCDEF" と同じ。

■ 3. 変数と型、代入・初期化
- int 型の変数を使って整数を扱う。
  例:
    int x;        // 宣言のみ（中身は不定値）
    x = 5;        // 代入
    int y = 10;   // 宣言と同時に初期化
- double 型の変数を使って小数を扱う。
- 文字は char 型、文字列は string 型で扱う。
- string の初期化と代入：
  string s1 = "ABC";
  s1 = "FBI";

■ 4. 入力（cin）と計算
- cin >> 変数; でキーボードから値を読み込む。
  例:
    int a, b;
    cin >> a >> b;
- 加減乗除・剰余演算：
  +, -, *, /, %
- 整数どうしの / は小数が切り捨てられる。
  例: (3 + 4) / 2 は 3 になる。
  小数を含めたいときは 2.0 などの実数を使う。

■ 5. 円周・面積と定数（const）
- 円周や面積の計算例：
  長さ ＝ 2 * π * r
  面積 ＝ π * r * r
- π をそのまま 3.14 と書く例と、const で定数オブジェクトにする例がある。
  例:
    const double PI = 3.1416;
- const をつけると値を変更できない定数になる。

■ 6. 乱数の基本
- #include <ctime>, #include <cstdlib> を使う。
- srand(time(NULL)); で乱数の種を初期化する。
- rand() % n で 0〜n-1 の乱数が得られる。
  例: rand() % 10 で 0〜9 の「ラッキーナンバー」を生成。

■ 7. 文字と文字列の入力
- char で1文字を読み込む：
  char c;
  cin >> c;
- string で単語（スペースまで）を読み込む：
  string name;
  cin >> name;
- getline(cin, s) で行全体（スペースを含む）を読み込む：
  string name;
  getline(cin, name);

■ 8. 練習問題・課題で扱ったポイント
- 固定の値（18と63など）の和・平均を計算して表示。
- 変数の初期化されていない状態（不定値）を確認。
- 整数の加減乗除・剰余の計算結果を表示。
- 実数の加減乗除の計算結果を表示。
- 符号反転の例：b = -a; として -a を計算。
- while 文とデクリメント演算子（x--, --x）の動きの確認。
- C++20 の std::format を使った整形出力（桁幅や小数点以下の桁数）。

■ 9. 用語の整理（基礎ドリルより）
- リテラル：ソースコード中に直接書かれた定数値（3.14, "C++", 'A' など）。
- エスケープシーケンス：\n（改行）、\t（タブ） など。
- main 関数：C++ プログラムで最初に呼ばれる関数。
- using namespace std; の利点と注意点：
  - 利点：std::cout を cout と書ける。
  - 注意点：名前の衝突が起きやすくなる。
- rand() の戻り値の範囲：0 〜 RAND_MAX の整数。

この第一章では、「画面への出力」「基本的な型と変数」「入力」「四則演算」「定数」「乱数」「文字・文字列の扱い」といった C++ の最初の基礎を一通り練習している。
