### 【レベル1：基礎理解】

**Q1.** 「式」とは何かを説明しなさい。
A. 式とは、演算子が入ってる文字の並びのこと。

正：式とは、評価すると値が得られる記述のこと。
たとえば、
a + b →　足した結果（値）が得られる
x > y →　比べた結果（true/false)が得られる

**Q2.** 「文」と「式」の違いを簡単に述べなさい。
A. 文とは、演算子が入ってない。式とは、演算子が入ってる。

正：
文は、処理（効果）を表す単位。
式は、値を生む。式は文の中に入れられる。

**Q3.** C++では、`bool`型の値として真と偽を表すキーワードをそれぞれ答えなさい。
A. 0か1かで、0だと偽、1だと真。

正：boolのキーワードはtrue/false。0/1は"数値としての評価"

**Q4.** 次のうち「偽」とみなされるものをすべて選びなさい。
　a) 0　　b) -1　　c) 2　　d) false
A. a, d

**Q5.** 「短絡評価」とは何かを簡潔に説明しなさい。
A. {}をなくして短く評価すること

正：短絡評価とは、and(&&)やor(||)の論理演算で、
左側の式の結果だけで全体の審議が決まるときに、右側の式を評価しない仕組みのこと
たとえば、
if (x != 0 && 10 / x > 1)
左がfalseの時、右の10 / xは評価されず、0除算を防げるんだね。

int a = 5;

a) A

### 【レベル3：switch文の理解】

**Q9.** switch文において `break;` を書かないとどうなるか。
A. break;と書かれるまで次の文も出力する。

△：fall-throughして次のcaseへ進む（「出力」ではなく「実行が続く」）。

**Q10.** 次のコードの出力を答えなさい。

```cpp
int n = 3;
switch (n) {
    case 1: cout << "one";
    case 2: cout << "two";
    case 3: cout << "three";
    default: cout << "done";
}
```

a) one
b) three
c) three done
d) one two three done

A. b

正：c（threeの後にdefaultでdone）（break;がないから）

**Q11.** `switch`文の`default`ラベルはどのようなときに実行されるか。
A. 入力された文字がswitch文の中でどれも当てはまらない場合に実行される

### 【レベル4：ブロックとスコープ】

**Q12.** ブロック `{ }` の中で宣言した変数はどの範囲で使えるか。
A. ブロック内の範囲全部使える
そのブロック内でのみ有効（スコープ）。

**Q13.** 次のプログラムを実行したときにエラーが出る理由を説明しなさい。

```cpp
if (true) {
    int x = 10;
}
cout << x;  // ここ
```

A. ""や''がない。正しくは、cout << "x";

正：エラー原因はスコープ外参照。xはブロックの外では未定義。

**Q14.** 以下のコードの出力を答えなさい。

```cpp
int a = 10, b = 20;
if (a > b) {
    int temp = a;
    a = b;
    b = temp;
}
cout << a << " " << b;
```

A. 10 20

### 【レベル5：応用・評価順】

**Q15.** 論理演算子 `&&` と `||` における「短絡評価」を説明し、次の式がどのように評価されるかを書きなさい。

```cpp
int x = 0;
if (x != 0 && 10 / x > 1)
    cout << "ok";
else
    cout << "no";
```

A. ifの条件に当てはまらないので出力はnoになる
左がfalseなので右（10 / x）は評価されずno。0除算を避けられるのがポイント。

**Q16.** 次のコードの実行結果を答えなさい。

```cpp
int x = 2, y = 3;
int z = (x > y) ? x : y;
cout << z;
```

A. 3

**Q17.** 条件演算子（`?:`）を使って、整数 `a` が偶数なら `"even"`、奇数なら `"odd"` を出力する1行のコードを書きなさい。
A. (a % 2) ? odd : even;

正：cout << ((a % 2 == 0) ? "even" : "odd");

### 【レベル6：総合問題】

**Q18.** 2つの整数を昇順に並べ替えるプログラムを if文を使って書きなさい。
A.

#include <iostream>

using namespace std;

int main()
{
    int a, b;

    cout << "変数a：";  cin >> a;
    cout << "変数b：";  cin >> b;

    if(a > b){
        int t = a;
        a = b;
        b = a; // 惜しい！正しくは：b = t;
    }

    cout << "変数を昇順に並び替えます。\n";
    cout << a << "\n";
    cout << b << "\n";
}

**Q19.** 変数 `score` の値に応じて次のように表示する switch文を書きなさい。

* 90〜100：`A`
* 80〜89：`B`

* それ以外：`F`

A.

#include <iostream>

using namespace std;

int main()
{
    int s;

    cout << "スコアを入力：";

        case 70 <= 79 : cout << "C";    break;
        default : cout << "F";          break;
    }

    /*
    正しくは

    switch (score / 10) {
        case 10 :
        case 9 : cout << "A";    break;
        case 8 : cout << "B";    break;
        case 7 : cout << "C";    break;
        default : cout << "F";   break;
    }
    */
}

### 【おまけ：記述問題】

**Q21.** 「式には値があり、文には効果がある」と言われる理由を説明しなさい。
A.式と文の違いを確かめるため
正：式は評価すると結果（値）が得られるのに対し、文はプログラムの状態を変える動作（効果）をもつためである。
たとえば、
a + bは計算結果という値を返す式であり、a = b;は代入という効果を実行する文である。
したがって「式には値があり、文には効果がある」と言われる。

**Q22.** 「switch文では変数を条件にできない場合がある」とはどういう意味か。
A.変数を条件にできないから
正：switch文の条件式には、整数型や列挙型しか使えないため、
文字列型や浮動小数点型などの変数を条件にするとエラーになる、という意味である。

**Q23.** `int` と `bool` の違いを、評価結果の観点から説明しなさい。
A.intは0かそれ以外か、boolは0か1か
正：intは多くの整数値を表せるのに対し、boolは真(true)か偽(false)の２つの値しか持たない。
条件式では、intは0がfalse、それ以外はtrueに変換される。
